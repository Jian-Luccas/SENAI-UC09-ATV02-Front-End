# SENAI-UC09-ATV02-Front-End
Atividade 2 – UC09 | Codificação para Front-End - Desenvolvimento da página index (Home) e Login de um site de e-commerce gamer, a partir de requisitos e prototipação. Projeto criado com HTML5, CSS3, Bootstrap, Media Queries, JavaScript e jQuery, com foco em responsividade, interatividade e experiência do usuário.

Após a definição dos requisitos e a criação dos protótipos de interface, este projeto tem como objetivo o desenvolvimento prático do Front-End de um site de e-commerce voltado ao universo gamer, utilizando layout para desktop.

Foram desenvolvidas as páginas Home (index) e Login, contendo os principais elementos de um site comercial, como menu de navegação, banner, seções de destaque e novidades, galeria de produtos e rodapé.

O projeto foi implementado aplicando as seguintes tecnologias e conceitos:

- HTML5 e CSS3 para a estruturação e estilização das páginas;
- Media Queries e Bootstrap para garantir a responsividade do layout;
- JavaScript e jQuery para implementação de funcionalidades e interações dinâmicas, como formulários, botões e componentes interativos;
- Uso de recursos como carrossel e mensagens interativas, visando melhorar a experiência do usuário e tornar a navegação mais intuitiva.

Este projeto faz parte de uma atividade prática de Front-End, com foco na aplicação de conceitos de UX, UI, responsividade e interatividade, simulando o desenvolvimento real de um site comercial.
